## Usage

Sparklines are built for **small, axis-free** trend strips — not a shrunken line chart. Unlike `<NQLineChart />`, which usually scales from **zero** and shows axes, legends, and brush, sparklines **pin the Y domain to your data min/max** (with a little padding) so variation fills the full height of a short container. Pass rows with a numeric `valueDataKey`, map series colors through `chartConfig`, and compose only the pieces you need — typically `<Sparkline />`, optionally `<Fill />`, `<EndDot />`, `<ReferenceBand />`, or `<Tooltip />`. Size the chart with `className` (for example `h-16 w-full`).

```tsx
import {
  NQSparklineChart,
  Sparkline,
  Fill,
  EndDot,
  Tooltip,
} from "@/components/nqchart/charts/sparkline-chart";
```

```tsx
const data = [{ value: 12 }, { value: 18 }, { value: 24 }];

const chartConfig = {
  trend: {
    label: "Sessions",
    colors: { light: ["#047857"], dark: ["#10b981"] },
  },
} satisfies ChartConfig;

<NQSparklineChart
  data={data}
  valueDataKey="value"
  config={chartConfig}
  backgroundVariant="dots"
  className="h-12 w-40"
>
  <Fill dataKey="trend" />
  <Sparkline dataKey="trend">
    <EndDot variant="colored-border" />
  </Sparkline>
  <Tooltip defaultIndex={data.length - 1} />
</NQSparklineChart>
```

## API Reference

<ApiHeading>NQSparklineChart</ApiHeading>

Root container for compact line charts. Renders a hidden Y axis with a **data-min / data-max** domain (not zero-based). No visible axes — consumers control footprint via `className`.

<ApiTable>
  <ApiRow name="config" type="ChartConfig" required>
    Series labels and theme-aware gradient colors keyed by each `<Sparkline dataKey />`.
  </ApiRow>
  <ApiRow name="data" type="TData[]" required>
    Rows containing the numeric field referenced by `valueDataKey`.
  </ApiRow>
  <ApiRow name="valueDataKey" type='keyof TData & string' required>
    Numeric column plotted on the sparkline (for example `"value"`).
  </ApiRow>
  <ApiRow name="children" type="ReactNode" required>
    Compose `<Fill />`, `<Sparkline />`, `<ReferenceBand />`, `<Tooltip />`.
  </ApiRow>
  <ApiRow name="className" type="string">
    Size and layout classes on `<ChartContainer />` (height/width for inline use).
  </ApiRow>
  <ApiRow name="curveType" type="CurveType" default='"monotone"'>
    Default interpolation inherited by `<Sparkline />` and `<Fill />`.
  </ApiRow>
  <ApiRow name="animationType" type='"none" | "left-to-right"' default='"left-to-right"'>
    Intro reveal mask; respects OS reduce-motion preference.
  </ApiRow>
  <ApiRow name="isLoading" type="boolean" default="false">
    Shows shimmer skeleton line instead of composed series.
  </ApiRow>
  <ApiRow name="loadingPoints" type="number" default="18">
    Number of placeholder points during loading.
  </ApiRow>
  <ApiRow name="backgroundVariant" type="BackgroundVariant">
    Draws `<ChartBackground />` behind the series (`dots`, `grid`, `bubbles`, etc.).
  </ApiRow>
  <ApiRow name="chartProps" type="ComponentProps<typeof LineChart>">
    Escape hatch for additional ECharts options (margins, `dataZoom`, etc.) compiled by NQChart.
  </ApiRow>
</ApiTable>

<ApiHeading>Sparkline</ApiHeading>

Single-series stroke with optional glow, dashed stroke, and terminal `<EndDot />`.

<ApiTable>
  <ApiRow name="dataKey" type="string" required>
    Key into `chartConfig` for gradients and tooltip labels.
  </ApiRow>
  <ApiRow name="strokeVariant" type='"solid" | "dashed"' default='"solid"'>
    Stroke dash pattern.
  </ApiRow>
  <ApiRow name="glowing" type="boolean" default="false">
    Applies soft SVG glow filter to the stroke.
  </ApiRow>
  <ApiRow name="children" type="ReactNode">
    Optional `<EndDot variant="..." />` configuration slot.
  </ApiRow>
</ApiTable>

<ApiHeading>Fill</ApiHeading>

Gradient area under the stroke (render **before** `<Sparkline />` so the line stays on top).

<ApiTable>
  <ApiRow name="dataKey" type="string" required>
    Config key shared with `<Sparkline />`.
  </ApiRow>
  <ApiRow name="fillOpacity" type="number" default="1">
    Opacity multiplier on the fill pattern.
  </ApiRow>
</ApiTable>

<ApiHeading>EndDot</ApiHeading>

Configuration slot — renders a `<ChartDot />` on the **last** point only when composed inside `<Sparkline />`.

<ApiTable>
  <ApiRow name="variant" type="DotVariant">
    Dot style forwarded to `@nqchart/dot`.
  </ApiRow>
</ApiTable>

<ApiHeading>ReferenceBand</ApiHeading>

Muted horizontal band for targets, SLA ranges, or baselines.

<ApiTable>
  <ApiRow name="y1" type="number" required>
    Lower bound in data units.
  </ApiRow>
  <ApiRow name="y2" type="number" required>
    Upper bound in data units.
  </ApiRow>
  <ApiRow name="fillOpacity" type="number" default="0.12">
    Band fill strength.
  </ApiRow>
</ApiTable>

<ApiHeading>Tooltip</ApiHeading>

Compact hover readout; cursor line disabled by default for inline charts.

<ApiTable>
  <ApiRow name="variant" type="TooltipVariant">
    Tooltip surface style.
  </ApiRow>
  <ApiRow name="cursor" type="boolean" default="false">
    Vertical crosshair while hovering.
  </ApiRow>
  <ApiRow name="defaultIndex" type="number">
    Highlights a point while idle (useful with `<EndDot />` on the last index).
  </ApiRow>
</ApiTable>
