## Usage

The funnel chart is composible. `<NQFunnelChart>` compiles a vertically laid out ECharts bar series with custom trapezoid shapes per stage (`stageKey`) and measured values (`valueKey`). Compose `<Grid />`, `<XAxis />`, `<YAxis />`, `<Stages />`, `<Tooltip />`, `<Legend />` as needed, and set optional `backgroundVariant` on the chart root — ECharts has no standalone funnel primitive, but additional options can pass through `chartProps` alongside the underlying <Link href="https://echarts.apache.org/en/option.html#series-bar" _blank>bar series</Link> escape hatch documented below.

```tsx
import {
  NQFunnelChart,
  Stages,
  XAxis,
  YAxis,
  Grid,
  Tooltip,
  Legend,
} from "@/components/nqchart/charts/funnel-chart";
```

```tsx
<NQFunnelChart
  config={chartConfig}
  data={funnelRows}
  stageKey="stage"
  valueKey="amount"
  backgroundVariant="dots"
>
  <YAxis />
  <XAxis />
  <Stages isClickable glowingStages={["trials"]} />
  <Grid />
  <Tooltip />
  <Legend isClickable />
</NQFunnelChart>
```

### Interactive Selection

Add `isClickable` to `<Stages />` (plus `<Legend />` when you want swatches clickable) then listen on `onSelectionChange` (`{ dataKey, value } | null`). `dataKey` matches rows’ `stageKey` string:

```tsx
<NQFunnelChart
  config={chartConfig}
  data={funnelRows}
  stageKey="stage"
  valueKey="amount"
  onSelectionChange={(selection) => {
    if (selection) {
      console.log("Selected:", selection.dataKey, selection.value);
    } else {
      console.log("Deselected");
    }
  }}
>
  <YAxis />
  <XAxis />
  <Stages isClickable />
  <Legend isClickable />
  <Tooltip />
</NQFunnelChart>
```

## Examples

### Intro animation

Funnel stages expand from the center on first paint (motion.dev — ECharts built-in bar animation is off). Default `animationType="top-to-bottom"` staggers from the first stage downward. Also `"bottom-to-top"`, `"center-out"`, `"edges-in"`, or `"none"`. Honors reduce motion.

```tsx
<NQFunnelChart animationType="center-out" ...>
```

## API Reference

The chart is composed of several parts. The props below are grouped by the component they belong to.

<ApiHeading>NQFunnelChart</ApiHeading>

The root container wrapping a vertical `<BarChart>`. Holds stage metadata, synthesized loading rows when `isLoading`, max-value scaling shared by funnel shapes, and selection state surfaced to `<Stages />`/`<Legend />`.

<ApiTable>
  <ApiRow name="config" type="Record<string, ChartConfig[string]>" required>
    Keys must correspond to serialized `stageKey` values (`string`) so gradients + legend labels resolve per funnel stage color stop.
  </ApiRow>
  <ApiRow name="data" type="TData[]" required>
    Rows with `stageKey` + numeric `valueKey` fields typed as literals for inference (`TData extends Record<string, unknown>`).
  </ApiRow>
  <ApiRow name="stageKey" type='keyof TData & string' required>
    Categorical funnel stage identifier surfaced on `<YAxis />` ticks and gradients.
  </ApiRow>
  <ApiRow name="valueKey" type='keyof TData & string' required>
    Numeric throughput used to scale each trapezoid width plus tooltip payload.
  </ApiRow>
  <ApiRow name="children" type="ReactNode" required>
    `<Grid />`, `<XAxis />`, `<YAxis />`, `<Stages />`, `<Tooltip />`, `<Legend />`, plus optional root `backgroundVariant`, and escapes through additional ECharts options via `chartProps`.
  </ApiRow>
  <ApiRow name="className" type="string">
    Additional wrapper classes forwarded to `<ChartContainer />`.
  </ApiRow>
  <ApiRow name="stageGap" type="number" default="6">
    Vertical gutter between stacked funnel polygons inside each category band.
  </ApiRow>
  <ApiRow name="backgroundVariant" type="BackgroundVariant">
    Draws `<ChartBackground />` beneath the funnel (same pattern as bar and scatter charts).
  </ApiRow>
  <ApiRow name="defaultSelectedStage" type="string | null" default="null">
    Pre-select matching `stageKey` string (`null` clears).
  </ApiRow>
  <ApiRow name="onSelectionChange" type="(selection: { dataKey: string; value: number } | null) => void">
    Fires when clickable legend / stage polygons toggle visibility focus.
  </ApiRow>
  <ApiRow name="isLoading" type="boolean" default="false">
    Swaps rendered rows with motion-shimmer placeholders while preserving axes + tooltip wiring.
  </ApiRow>
  <ApiRow name="loadingStages" type="number" default="5">
    Count of synthesized placeholder polygons while `isLoading` is active.
  </ApiRow>
  <ApiRow name="animationType" type='"none" | "top-to-bottom" | "bottom-to-top" | "center-out" | "edges-in"' default='"top-to-bottom"'>
    Staggered center-out grow for each trapezoid stage on first render.
  </ApiRow>
  <ApiRow name="chartProps" type="ComponentProps<typeof BarChart>">
    Additional ECharts options merged into the compiled `option` by NQChart — sizing, responsiveness, `dataZoom`, margins, and more. Refer to <Link href="https://echarts.apache.org/en/option.html#series-bar" _blank>ECharts bar series</Link>; there is no dedicated funnel primitive, so this escape hatch doubles as funnel tuning.
  </ApiRow>
</ApiTable>

<ApiHeading>Stages</ApiHeading>

Configuration slot mirrored by funnel renderer — determines whether polygons listen for clicks / glow defs.

<ApiTable>
  <ApiRow name="isClickable" type="boolean" default="false">
    Enables polygon click toggling. Selection dimming applies whenever a stage is selected (legend, default, or click).
  </ApiRow>
  <ApiRow name="glowingStages" type="string[]" default="[]">
    Names matching `stageKey` values emitting glow filters akin to NQBars.
  </ApiRow>
</ApiTable>

<ApiHeading>XAxis</ApiHeading>

Mirrors numeric scale even though NQFunnel hides ticks by default to keep silhouette clean (`hide` maps to ECharts axis visibility).

<ApiTable>
  <ApiRow name="hide" type="boolean" default="true">
    Default hides axis line / ticks consistent with NQFunnels marketing aesthetic.
  </ApiRow>
  <ApiRow name="type" type='"number" | ...' default='"number"'>
    Forwarded verbatim to `<XAxis />` while loading states hide axes automatically.
  </ApiRow>
  <ApiRow name="…axisProps">
    Additional xAxis options are merged into the compiled ECharts option. Read the <Link href="https://echarts.apache.org/en/option.html#xAxis" _blank>ECharts xAxis</Link> documentation for available props.
  </ApiRow>
</ApiTable>

<ApiHeading>YAxis</ApiHeading>

Shows ordered funnel stages (category axis for vertical funnel layout).

<ApiTable>
  <ApiRow name="width" type="number" default="96">
    Default tick gutter width accommodating stage labels before trapezoid math runs.
  </ApiRow>
  <ApiRow name="type" type='"category" | ...' default='"category"'>
    Drives categorical placement for each funnel row (`dataKey={stageKey}` internally).
  </ApiRow>
  <ApiRow name="…axisProps">
    Additional yAxis options are merged into the compiled ECharts option. Read the <Link href="https://echarts.apache.org/en/option.html#yAxis" _blank>ECharts yAxis</Link> documentation for available props.
  </ApiRow>
</ApiTable>

<ApiHeading>Grid</ApiHeading>

Optional vertical-only dashed scaffolding for alignment reference.

<ApiTable>
  <ApiRow name="strokeDasharray" type="string" default='"3 3"'>
    Dash pattern forwarded to `<CartesianGrid />`.
  </ApiRow>
  <ApiRow name="…gridProps">
    Horizontal lines stay disabled intentionally; remaining grid options are merged into the compiled ECharts option. Read the <Link href="https://echarts.apache.org/en/option.html#grid" _blank>ECharts grid documentation</Link> for nuance.
  </ApiRow>
</ApiTable>

<ApiHeading>Tooltip</ApiHeading>

Disables Cartesian cursor crosshair (`cursor={false}`) for cleaner funnel hovering.

<ApiTable>
  <ApiRow name="variant" type='"default" | "frosted-glass"' default='"default"'>
    Tooltip polish tier.
  </ApiRow>
  <ApiRow name="roundness" type='"sm" | "md" | "lg" | "xl"' default='"lg"'>
    Tooltip radius.
  </ApiRow>
  <ApiRow name="defaultIndex" type="number">
    Pre-highlights tooltip for a deterministic stage row while pointer idle.
  </ApiRow>
</ApiTable>

<ApiHeading>Legend</ApiHeading>

Stage keyed legend syncing dimming + selection mirrored from `<Stages />`.

<ApiTable>
  <ApiRow name="variant" type='"square" | "circle" | "circle-outline" | "rounded-square" | "rounded-square-outline" | "vertical-bar" | "horizontal-bar"'>
    Legend marker motifs.
  </ApiRow>
  <ApiRow name="align" type='"left" | "center" | "right"' default='"center"'>
    Horizontal alignment.
  </ApiRow>
  <ApiRow name="verticalAlign" type='"top" | "middle" | "bottom"' default='"bottom"'>
    Vertical stacking vs chart canvas.
  </ApiRow>
  <ApiRow name="isClickable" type="boolean" default="false">
    Legend entries toggle shared selection; dimming follows `selectedStage` even when `<Stages />` is not clickable.
  </ApiRow>
</ApiTable>
