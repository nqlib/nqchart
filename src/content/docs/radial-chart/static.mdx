## Usage

The radial chart is composible. `<BeeRadialChart>` is the container, and you compose only the parts you need — `<Legend>`, `<Tooltip>`, and a `<RadialBar>` — as its children. The `<RadialBar>` carries its own `glowingBars` and `isClickable`, so styling and interactivity live with the series itself.

```tsx
import {
  BeeRadialChart,
  RadialBar,
  Tooltip,
  Legend,
} from "@/components/beecharts/charts/radial-chart";
```

```tsx
const data = [
  { browser: "chrome", visitors: 275 },
  { browser: "safari", visitors: 200 },
  { browser: "firefox", visitors: 187 },
];

const chartConfig = {
  chrome: {
    label: "Chrome",
    colors: { light: ["#3b82f6"], dark: ["#60a5fa"] },
  },
  safari: {
    label: "Safari",
    colors: { light: ["#10b981"], dark: ["#34d399"] },
  },
  firefox: {
    label: "Firefox",
    colors: { light: ["#f59e0b"], dark: ["#fbbf24"] },
  },
} satisfies ChartConfig;
```

```tsx
<BeeRadialChart data={data} nameKey="browser" config={chartConfig} variant="full">
  <Legend isClickable />
  <Tooltip />
  <RadialBar dataKey="visitors" isClickable glowingBars={["chrome"]} />
</BeeRadialChart>
```

### Interactive Selection

Add `isClickable` to `<RadialBar>` (and to `<Legend>`) to make bars selectable. Use the `onSelectionChange` callback on `<BeeRadialChart>` to handle selection events:

```tsx
<BeeRadialChart
  data={data}
  nameKey="browser"
  config={chartConfig}
  onSelectionChange={(selection) => {
    if (selection) {
      console.log("Selected:", selection.dataKey, "Value:", selection.value);
    } else {
      console.log("Deselected");
    }
  }}
>
  <Legend isClickable />
  <Tooltip />
  <RadialBar dataKey="visitors" isClickable />
</BeeRadialChart>
```

### ECharts radial patterns

BeeCharts maps ECharts polar families to primitives (no duplicate packages):

| Pattern | BeeCharts | When |
| --- | --- | --- |
| Concentric rings (arc = value) | `BeeRadialChart` default | Compare metrics per category on separate rings |
| Rose / nightingale (petal length = value) | `BeeRadialChart` `layout="rose"` | Cyclical categories (hours, months, directions) |
| KPI gauge (single row) | `BeeRadialChart` `variant="semi"` + one row | Scorecards, NPS, progress |
| Pie / donut slices | [`BeePieChart`](/docs/pie-chart/static) | Part-to-whole on one ring |
| Spider / radar polygon | [`BeeRadarChart`](/docs/radar-chart/static) | Multi-axis profiles |
| Nested hierarchy | [`BeeTreemapChart`](/docs/treemap-chart/static) | Category breakdown (rectangular; sunburst is not bundled yet) |

Intro animation: all rings or petals sweep **in sync** (~900ms clockwise).

## API Reference

The chart is composed of several parts. The props below are grouped by the component they belong to.

<ApiHeading>BeeRadialChart</ApiHeading>

The root container. It owns the data, the shared selection state, the loading skeleton, and the chart-wide arc shape. Everything visual is composed as its children.

<ApiTable>
  <ApiRow name="data" type="TData[]" required>
    Data used to display the chart. An array of objects where each object represents a radial bar (`TData extends Record<string, unknown>`).
  </ApiRow>
  <ApiRow name="config" type="ChartConfig" required>
    Configuration object that defines the chart's bars. Each key should match a value from your `nameKey` field, with corresponding colors.
  </ApiRow>
  <ApiRow name="nameKey" type="keyof TData & string" required>
    The key from your data objects to use for bar names (typically string values used for labels and the legend).
  </ApiRow>
  <ApiRow name="children" type="ReactNode" required>
    The composed chart parts — `<Legend />`, `<Tooltip />`, and a `<RadialBar />`.
  </ApiRow>
  <ApiRow name="className" type="string">
    Additional CSS classes to apply to the chart container.
  </ApiRow>
  <ApiRow name="variant" type='"full" | "semi"' default='"semi"'>
    The arc shape of the radial chart. `"full"` displays a full circle (360°), `"semi"` displays a half circle (180°).
  </ApiRow>
  <ApiRow name="layout" type='"concentric" | "rose"' default='"concentric"'>
    Polar layout. `"concentric"` — one ring per row, arc length is the value. `"rose"` — category petals radiating from the center (nightingale rose).
  </ApiRow>
  <ApiRow name="innerRadius" type="number | string" default='"30%"'>
    The inner radius of the radial bars. Can be a number (pixels) or percentage string.
  </ApiRow>
  <ApiRow name="outerRadius" type="number | string" default='"100%"'>
    The outer radius of the radial bars. Can be a number (pixels) or percentage string.
  </ApiRow>
  <ApiRow name="defaultSelectedDataKey" type="string | null" default="null">
    The bar name that should be selected by default.
  </ApiRow>
  <ApiRow name="onSelectionChange" type="(selection: { dataKey: string; value: number } | null) => void">
    Callback fired when a bar is selected or deselected — by clicking a clickable `<RadialBar />` or `<Legend />` entry. Receives an object with `dataKey` (bar name) and `value` (bar value), or `null` when deselected.
  </ApiRow>
  <ApiRow name="isLoading" type="boolean" default="false">
    Shows a loading placeholder animation when data is being fetched.
  </ApiRow>
  <ApiRow name="backgroundVariant" type="BackgroundVariant">
    Background pattern variant to display behind the chart.
  </ApiRow>
  <ApiRow name="chartProps" type="ComponentProps<typeof RadialBarChart>">
    Additional ECharts options merged into the compiled `option` by BeeCharts. Read the <Link href="https://echarts.apache.org/en/option.html#series-bar" _blank>ECharts RadialBarChart documentation</Link> for available props.
  </ApiRow>
</ApiTable>

<ApiHeading>RadialBar</ApiHeading>

The radial bar series. Each data row becomes one bar. It generates its own glow filter definitions, so glow effects never collide with other charts on the page.

<ApiTable>
  <ApiRow name="dataKey" type="string" required>
    The key from your data objects to use for bar values (typically numeric values that determine bar size).
  </ApiRow>
  <ApiRow name="cornerRadius" type="number" default="5">
    The border radius for the corners of each bar in pixels.
  </ApiRow>
  <ApiRow name="barSize" type="number" default="14">
    The thickness of each radial bar in pixels.
  </ApiRow>
  <ApiRow name="showBackground" type="boolean" default="true">
    Whether to render the background track (the unfilled portion of each bar).
  </ApiRow>
  <ApiRow name="isClickable" type="boolean" default="false">
    Enables interactive clicking on bars to select/deselect them. Unselected bars dim while a selection is active.
  </ApiRow>
  <ApiRow name="glowingBars" type="string[]" default="[]">
    Array of bar names (values from your `nameKey` field) that should have a glowing effect applied. Creates a smooth outer glow around the specified bars.
  </ApiRow>
  <ApiRow name="radialBarProps" type='Omit<ComponentProps<typeof RadialBar>, "dataKey">'>
    Additional ECharts options merged into the compiled `option` by BeeCharts. Read the <Link href="https://echarts.apache.org/en/option.html#series-bar" _blank>ECharts RadialBar documentation</Link> for available props.
  </ApiRow>
</ApiTable>

<ApiHeading>Tooltip</ApiHeading>

The hover tooltip. Labels each bar by its name. Render it to show a tooltip, omit it for none.

<ApiTable>
  <ApiRow name="variant" type='"default" | "frosted-glass"' default='"default"'>
    Controls the visual style of the tooltip.
  </ApiRow>
  <ApiRow name="roundness" type='"sm" | "md" | "lg" | "xl"' default='"lg"'>
    Controls the border-radius of the tooltip.
  </ApiRow>
  <ApiRow name="defaultIndex" type="number">
    When set, the tooltip will be visible by default at the specified data point index.
  </ApiRow>
</ApiTable>

<ApiHeading>Legend</ApiHeading>

The bar legend. When `isClickable` is set, each entry toggles selection of its bar. Render it to show a legend, omit it for none.

<ApiTable>
  <ApiRow name="variant" type='"square" | "circle" | "circle-outline" | "rounded-square" | "rounded-square-outline" | …'>
    The visual style variant for the legend indicators.
  </ApiRow>
  <ApiRow name="align" type='"left" | "center" | "right"' default='"center"'>
    Horizontal placement of the legend.
  </ApiRow>
  <ApiRow name="verticalAlign" type='"top" | "middle" | "bottom"' default='"bottom"'>
    Vertical placement of the legend.
  </ApiRow>
  <ApiRow name="isClickable" type="boolean" default="false">
    When enabled, each legend entry toggles selection of its bar, driving the shared selection state read by `<RadialBar />`.
  </ApiRow>
</ApiTable>
